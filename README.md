#  WhatsApp Chat Analytics Dashboard

A professional data visualization dashboard built using **Python, Pandas, and Matplotlib** to analyze messaging activity and communication patterns.

This project simulates analytics that could be used in messaging platforms such as WhatsApp, Slack, or customer support systems.

---

## 🚀 Project Features

The dashboard visualizes the following insights:

- Messages per User
- Weekly Activity Analysis
- Sentiment Distribution
- Hourly Chat Activity Heatmap
- Daily Chat Timeline
- Most Used Messages



##  Concepts Demonstrated

This project demonstrates multiple important **Data Science skills**:

- Data generation and simulation
- Feature engineering
- Time-series analysis
- Advanced visualization using Matplotlib
- Dashboard design using GridSpec
- Data storytelling


## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- NumPy
